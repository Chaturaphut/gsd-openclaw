# Discussion Log

> **Purpose:** Record all significant decisions, clarifications, and discussions during a GSD workflow.
> **Rule:** Every non-trivial decision gets an entry. Future agents read this to understand WHY, not just WHAT.

---

## Format

Each entry follows this structure:

```
### [DISC-NNN] Topic
- **Date:** YYYY-MM-DD
- **Participants:** [who was involved]
- **Context:** [what prompted the discussion]
- **Options Considered:**
  1. Option A — [pros/cons]
  2. Option B — [pros/cons]
- **Decision:** [what was decided]
- **Rationale:** [why this option was chosen]
- **Impact:** [what this changes in the plan/code]
```
